# Git Task Pack

This directory contains ready-to-use instruction files for the Git exercises.

How to use it:
- each file describes one task or one larger exercise block,
- commands are listed in the order you should run them,
- nothing here runs automatically,
- whenever you see a placeholder like `YOUR_REPO_URL`, replace it with your real data.

Notes:
- the commands are written so they work in Git Bash or PowerShell,
- some files include warnings for `git reset --hard` and `git push --force`,
- GitHub-related tasks include placeholders for repository names, branch names, and URLs.
